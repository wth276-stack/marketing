# Jessi Beauty Marketing 營運系統（逐步手冊）

> 外判 marketing 逐步手冊。每一步都有「準備 / 做 / 完成標準 / 時間」。
> **每日/每週節奏** → `每月營運時間表（可循環）.md`（master）。
> **同店分工 / SLA / Lead 交接** → `客戶配合流程`。
> 行動計劃（START HERE）＝大方向；呢份＝逐步點落手。
> 每步格式：**準備** = 做之前要 ready 咩；**做** = 逐個動作；**完成** = 點先叫做完。

---

# A. 一次性準備（開始前 set 好，慳之後所有時間）

## A1. 器材（拍片用）
- [ ] **手機**：清潔鏡頭；設定影片 1080p/4K、60fps（穩定）
- [ ] **三腳架／手機夾**（固定機位，唔郁手）
- [ ] **補光**：環形燈或近窗自然光（面光要均勻、唔好背光）
- [ ] **收音**：安靜房間，或夾一個細蜜蜂咪
- [ ] **場地 3 個位**：接待 / 分析房 / 儀器位（拍 B-roll 用）
- [ ] **老闆造型**：乾淨、貼品牌感（米白／淺金系）

## A2. 帳號／工具（一次過開好）
- [ ] IG 已轉**專業帳號**（Business）— 設定 → 帳號類型
- [ ] 開 **Meta Business Suite**（business.facebook.com）連住 IG → 用嚟排程
- [ ] **Canva** 帳號（整 Carousel／封面）
- [ ] 5 份檔案放埋一齊（知識庫／Prompt 包／內容日程／營運 SOP／tracker）
- [ ] `beauty-salon-marketing-tracker.html` 加書籤

## A3. 內容資產（一次過備好）
- [ ] Canva 整一個**品牌模板**：1080×1350、米白＋淺金、固定字體（之後 Carousel 套用）
- [ ] 諗定 5 個 DM 關鍵字嘅 auto-reply（分析／暗黃／泛紅／痘印／底妝唔貼）
- [ ] 客人同意書範本（見證／對比相用）

> ✅ 完成 A 之後，之後每週只需要「執行」，唔使再 set。

# B. Phase 0：帳號門面設定（做一次，逐步）

## 步驟 B1｜改 IG 名稱欄（⏱5 分）
- **準備**：揀定名稱欄文字（建議「JESSI 尖沙咀底色重整｜Jessi Glow·暗黃泛紅·底妝唔貼」）
- **做**：
  1. 開 IG → 個人檔案 → 「編輯個人檔案」
  2. 改「**名稱**」欄（唔係 username）
  3. 貼新文字 → 儲存
- **完成**：搜尋「尖沙咀底色重整」「Jessi Glow」見到你個 account 出現機會升

## 步驟 B2｜改 Bio（⏱10 分）
- **準備**：copy `營運 SOP` 第 2.2 節嘅 Bio 範本
- **做**：
  1. 編輯個人檔案 → 「簡介」
  2. 貼範本（化妝都唔乾淨／底色重整／免費皮膚分析／DM『分析』）
  3. Link：放 WhatsApp 或 Linktree → 儲存
- **完成**：Bio 有「免費皮膚分析」＋「DM『分析』」，冇「告別」「保證」等字

## 步驟 B3｜設 DM Auto-reply（⏱15 分）
- **準備**：copy `品牌知識庫` 第 5 節 3 條自測問題
- **做**：
  1. Meta Business Suite → 收件匣 → 自動回覆 / 常用語
  2. 設關鍵字（分析／暗黃／泛紅／痘印／底妝唔貼）觸發 → 自動回 3 條自測問題＋「客服稍後跟返你」
- **完成**：自己 DM 個關鍵字測試，收到自動回

## 步驟 B4｜整 Highlights（⏱30 分，可分次）
- **準備**：揀／影幾張封面圖（品牌色）
- **做**：逐個開 Highlight：免費分析 / 客人分享 / 底色自測 / Jessi Glow（＋保留路線圖；Focus Dual / 淡斑納入輔助）
- **完成**：6 個以內、命名清晰、有封面

## 步驟 B5｜Pin 3 條（⏱5 分，有內容後做）
- **做**：揀 1 條共鳴 Reel、1 條顯老 3 個位 Carousel、1 條免費分析介紹 → 長按貼文 → 釘選
- **完成**：profile 頂三格係呢三條

## 步驟 B6｜tracker 試行（⏱10 分）
- **做**：開 `tracker` → 撳「載入示例」睇下 funnel/圖表 → 撳「清空資料」
- **完成**：你知點睇 4 個數字同 funnel

> ✅ B 做完＝門面唔再漏客，可以開始出內容。

# C. 每週生產線（7 個工序，逐步）

> 第一個月：工序 2 用現成內容（`第一個月內容日程`），跳過 engine。
> 第二個月起：工序 2 用 engine（Prompt 1–4）。

## 工序 1｜揀方向（星期一・⏱45–90 分）
- **準備**：開 `tracker`；開 `workflow.html` 主題發掘
- **做**：
  1. 睇上週內容成效 + 店內週報 FAQ
  2. 將新 idea 加入 **backlog**
  3. **M2+**：揀本週主題（新鮮優先；同題材隔 8 週）/**M1**：抄內容日程
  4. 行 Prompt 1–4（M2+）或抄腳本（M1）
- **完成**：本週主題一句 + Brief 已填

## 工序 2A｜出內容（第一個月・⏱5 分）
- **做**：開 `第一個月內容日程`，揀今週（W?）→ copy 嗰 5 條內容
- **完成**：手上有 3 Reel 腳本＋1 Carousel＋Stories

## 工序 2B｜出內容（第二個月起・⏱45 分）
- **準備**：開新 AI chat
- **做**：
  1. 成份貼 `品牌知識庫`
  2. 行 `Prompt 1`（出計劃表）→ 睇 OK
  3. 行 `Prompt 2`（逐條生成）
  4. 行 `Prompt 3`（Polish 廣東話）
  5. 行 `Prompt 4`（合規自檢）
- **完成**：得到 Polish + 合規後嘅一週內容

## 工序 3｜拍攝（星期二・每 2 週半日）
- **準備**：A1 器材 ready；印／開腳本；對 `營運 SOP` 第 3 節整 Shot List
- **做**（每條 Reel）：
  1. set 三腳架＋補光，試一段睇光同聲
  2. 先拍 **Hook 特寫**（老闆講第一句，2–3 次取最好）
  3. 拍主體（旁白分段拍，唔使一 take 過）
  4. 拍 **B-roll**（前鏡、側面下顎線、分析房、儀器）
  5. 即場 review，唔清就補拍
- **做**（Stories／Carousel 素材）：影幾張環境、過程、細節相
- **完成**：3 Reel 片段 + Carousel 用圖 + Stories 素材，存入手機相簿一個 folder

## 工序 4｜後製 + Carousel（星期三・⏱3–4 小時，同剪 Reel 一齊做）
- **準備**：Canva 品牌模板（A3）；copy 內容檔嘅 Carousel 逐頁文字
- **做**：
  1. Canva 開模板 → 揀 1080×1350
  2. 逐頁貼標題＋內文（第 1 頁最搶眼、最後頁 CTA）
  3. 落圖／品牌色
  4. 下載 PNG（順序命名 1,2,3…）
- **完成**：一套 Carousel 圖 ready

## 工序 5｜排程（星期三一併完成；星期四前確認已排晒本週）
- **準備**：片剪好（加字幕）、Carousel 圖、caption（含 IG SEO keyword）
- **做**：
  1. Meta Business Suite → Planner → 建立貼文
  2. **Reels**：上載片 → caption（含 keyword）→ 揀封面 → 排時間；想測 hook 嘅用 IG app 內 **Trial Reels**（先推非 followers）
  3. **Carousel**：上載多張 → caption → alt text → 排時間
  4. **Stories**：可即日用手機出（投票／問答 sticker 要 in-app 加）
- **完成**：今週內容已排晒／設好提醒

## 工序 6｜互動 ＋ DM 成交（每日・⏱20–30 分）
- **準備**：開 `營運 SOP` 第 1 節話術
- **做**：
  1. 回有內容嘅留言（唔好齋回「多謝」）
  2. 客人 send 關鍵字 → 回 3 條自測問題（B3 已自動）
  3. 客人答 → 按 `SOP 1.2` 分類 → 用話術邀**免費皮膚分析**
  4. 約時間時，攞聯絡**前先攞同意**（SOP 1.4 私隱）
  5. 即刻去 tracker 開 lead；要店內 confirm 時段 → **Lead 交接**（`客戶配合流程` §7）
- **完成**：當日 DM 清零，新查詢入晒 tracker；已交接 lead 有回覆狀態

## 工序 7｜記數據 ＋ 復盤（星期五・⏱30–45 分）
- **準備**：開 `tracker`；開 IG 每條貼文嘅 Insights
- **做**：
  1. 「內容成效」section → 逐條填：Reach／完播%／Shares／Saves／DM／預約／成交／下次動作
  2. 「記錄」section → update lead 狀態（預約／到店／成交／流失原因）
  3. 撳「生成每週復盤」睇摘要
  4. 對 `營運 SOP` 第 4 節決策規則，寫低下週 keep / stop / 改
- **完成**：tracker up to date＋下週方向已定

# D. 月度循環（每月做一次）

## 步驟 D1｜月檢（⏱30 分）
- **做**：tracker 睇成個月——邊個痛點帶最多 DM／成交、邊個渠道最好
- **完成**：決定下個月加重邊個主題

## 步驟 D2｜下月主題方向（M2+）
- 睇邊類**主題/角度**帶最多 DM/成交（唔係邊個痛點輪到）
- 將有效方向拆成下月 backlog；停低無效題材
- **完成**：backlog 有至少 4 週候選題
